Week_2: Short Notes

Lecture1: Sets and Functions

Basic Sets & Notation

  • Real Numbers: R = all real values
  • Non-negative Reals: R⁺ = {x ∈ R | x ≥ 0}
  • Integers: Z = {…,−2,−1,0,1,2,…}
  • Non-negative Integers: Z⁺ = {x ∈Z | x ≥ 0}

Intervals

  • Closed Interval: [a,b] = {x ∈ R | a ≤ x ≤ b}
  • Open Interval: (a,b) = {x ∈ R | a < x < b}

Higher Dimensions

  • Rd: d-dimensional vectors, e.g. R³ = (x₁, x₂, x₃)
  • [a,b]d: each component lies in [a,b]

Metric Spaces (Distance)

Euclidean Distance:

d(x,y) = √( (x₁−y₁)² + ⋯ + (xd−yd)² )

Balls

  • Open Ball: B(x,ϵ) = {y | d(x,y) < ϵ}
  • Closed Ball: B̄(x,ϵ) = {y | d(x,y) ≤ ϵ}
  • Open = boundary excluded
  • Closed = boundary included

Set Operations & Logic

Operations

  • Union: A∪B → in A or B
  • Intersection: A∩B → in both
  • Complement: Ac = V∖A

De Morgan's Laws: (A∪B)c = Ac∩Bc, (A∩B)c = Ac∪Bc

Logic Symbols

  • ∀ → for all
  • ∃ → there exists
  • ⇒ → implies
  • ⇔ → equivalent

Sequences & Convergence

Definition: A sequence xn→x* if:

For every ϵ>0, ∃N such that for all n ≥ N: xn ∈ B(x*, ϵ)

Meaning: eventually stays close to the limit.

Examples

  • Converging: xn = (1 + 2/n, 3 − 2/n) → (1,3)
  • Non-Converging: (cos(2πn), sin(2πn)) → cycles, no limit

Vector Spaces

Definition: If u, v ∈ V and scalars α, β ∈ R, then: αu + βv ∈ V. Closed under linear combination.

Dot Product: x·y = Σi=1d xiyi

Norm (Length): ‖x‖₂ = √(x·x) = √( Σi=1d xi² )

Orthogonality

x·y = 0 ⇒ vectors are perpendicular

Functions & Visualization

Function: f:A→B — Maps input → output.

Types

  • Real-valued: f:Rd→R
  • Multivariate: multiple inputs

Graph: Gf = {(x, f(x))}

Visualization

  • 1D Function: f(x) = x² → parabola
  • 2D Functions (f:R²→R):
    • Contour Plot: lines of constant value
    • Heat Map: color = value

Examples:

  1. Linear: f(x) = x₁ + x₂ → contours = straight lines
  2. Multiplicative: f(x) = x₁x₂ → contours = hyperbolas
  3. Quadratic: f(x) = x₁² + x₂² → contours = circles

Quick Summary

  • Distance → Euclidean norm
  • Convergence → "eventually stays close"
  • Vector space → closed under linear combinations
  • Dot product → measures alignment
  • Orthogonal → dot product = 0
  • Contours → constant function values

Lecture2: Univariate Calculus: Continuity and Differentiability

Continuity

Intuitive Definition: A function is continuous at a point x* if:

  • Small changes in input ⇒ small changes in output
  • No jumps, breaks, or holes

Formal Definition: limx→x* f(x) = f(x*)

Equivalent sequence definition: If xn → x*, then f(xn) → f(x*).

Global Continuity: A function is continuous at every point in its domain.

Examples (Continuous Functions)

  1. Polynomial: f(x) = x² — Continuous everywhere. Example: as x → 2, f(x) → 4.
  2. Piecewise Continuous: f(x) = { 2x+1, x>1 ; 3, x≤1 }
    At x=1: left = 3, right = 3 → matches → continuous.

Examples (Discontinuous Functions)

  1. Sign Function: sgn(x) = { −1, x<0 ; 0, x=0 ; 1, x>0 }
    At x=0: left → −1, right → +1. Not continuous.
  2. Reciprocal Function: f(x) = 1/x — At x=0: undefined. Not continuous.
  3. Oscillating Function: f(x) = cos(1/x) — As x→0: no single limit. Not continuous.

Differentiability

Definition: A function is differentiable at x* if the following limit exists:

f′(x*) = limx→x* [ f(x)−f(x*) ] / (x−x*)

This value is called the derivative.

Relation with Continuity

  • Not continuous ⇒ not differentiable
  • Continuous ⇏ differentiable
  • Differentiability is a stronger condition than continuity

Examples (Not Differentiable)

  1. Absolute Value: f(x) = |x| — At x=0: left slope = −1, right slope = +1. Not differentiable.
  2. Piecewise Function: f(x) = { 4x+2, x≥2 ; 2x+6, x<2 }
    At x=2: left slope = 2, right slope = 4. Not differentiable.

Geometric Interpretation of Derivative

Alternative Formula: f′(x*) = limh→0 [ f(x*+h)−f(x*) ] / h

Meaning: The derivative is the slope of the tangent line at x*.

Intuition

  • Take two points on the curve.
  • Draw the secant line.
  • Move points closer.
  • As h→0, slope → exact slope at that point.

Linear Approximation: Near x*, the function behaves like: f(x) ≈ f(x*) + f′(x*)(x − x*)

Locally, the curve is approximated by a straight line.

Quick Summary

  • Continuity: no jumps, breaks, or holes
  • Differentiability: smooth curve, no sharp corners
  • Differentiable ⇒ always continuous
  • Not continuous ⇒ never differentiable
  • Derivative = slope of tangent line

Lecture3: Univariate Calculus: Derivatives and Linear Approximations

Core Concepts

Derivative (Idea)

  • Measures the rate of change or slope at a point.
  • Tells how fast a function changes near x*.

Linear Approximation (Main Idea)

  • Replace a complex function with a straight line near a point.
  • Works locally (near x*).
  • Exact at x = x*, good near x*, inaccurate far away.

Functional vs Geometric View

Functional (Algebra View): Linear approximation is a function of the form: y = mx + c, m = f′(x*)

Geometric View: It is the tangent line to the curve, touching the graph at (x*,f(x*)).

Key Formulae

Derivative Definition: f′(x*) = limx→x* [ f(x)−f(x*) ] / (x−x*)

Linear Approximation Formula: f(x) ≈ f(x*) + f′(x*)(x−x*)

Linear Function Notation: Lx*f(x) = f(x*) + f′(x*)(x−x*)

Classic Approximations (Near x=0)

FunctionApproximationReason
sin xxf(0) = 0, f′(0) = 1
ex1 + xf(0) = 1, f′(0) = 1
log(1 + x)xf(0) = 0, f′(0) = 1
(1 + x)r1 + rxderivative = r(1+x)r−1

Worked Examples

Example 1: Approximate f(x) = x² near x* = 1

  • f(1) = 1
  • f′(x) = 2x ⇒ f′(1) = 2
  • Linear approximation: L₁f(x) = 1 + 2(x−1) = 2x − 1

Final Answer: 2x−1

Example 2: Estimate 0.99⁷

Step 1: Rewrite 0.99⁷ = (1−0.01)⁷

Step 2: Compare with (1+x)r: x=−0.01, r=7

Step 3: Apply Approximation: (1+x)r ≈ 1 + rx
0.99⁷ ≈ 1+7(−0.01) = 1−0.07 = 0.93

Closest Answer: 0.93

Importance in Machine Learning

  • Used in optimization algorithms (e.g., gradient descent).
  • Converts complex functions into linear approximations for easier computation.
  • Helps in finding minima/maxima and training models efficiently.

Quick Summary

  • Derivative = slope at a point.
  • Linear approximation = tangent line.
  • Formula: f(x) ≈ f(x*) + f′(x*)(x−x*)
  • Accurate only near the point.
  • Core idea behind optimization in machine learning.

Lecture4: Univariate Calculus: Applications and Advanced Rules

Higher-Order Approximations

Idea

  • Linear approximation uses only slope (first derivative).
  • Higher-order approximations use curvature (second derivative, etc.).
  • More accuracy, but more complexity.

Linear Approximation (First Order): f(x) ≈ f(x*) + f′(x*)(x−x*)

Quadratic Approximation (Second Order): f(x) ≈ f(x*) + f′(x*)(x−x*) + 1/2 f″(x*)(x−x*)²

Key Insight

  • These are truncated Taylor series.
  • Quadratic approximation is more accurate than linear.
  • Exact for quadratic functions.

Example: ex near x = 0:

  • Linear: 1+x
  • Quadratic: 1 + x + x²/2 → better approximation.

Example: 1.1⁷

  • Linear: 1 + 7(0.1) = 1.7
  • Quadratic: 1 + 0.7 + (0.7)²/2 ≈ 1.91 → much closer to actual value.

Rules from Linear Approximation

Product Rule: If f(x) = g(x)·h(x), then: f′(x) = g′(x)h(x) + h′(x)g(x)
Idea: Approximate both functions, multiply, ignore higher-order terms.

Chain Rule: If f(x) = g(h(x)), then: f′(x) = g′(h(x))·h′(x)
Idea: First approximate inner function, then outer function.

Example: Approximate f(x) = e³ˣ/√(1+x) near x=0

  • e³ˣ ≈ 1+3x
  • (1+x)−1/2 ≈ 1−x/2
  • Multiply: (1+3x)(1−x/2) ≈ 1 + 5/2 x

Final Approximation: 1+5/2 x

Example: f(x) = e1+x near x=1 — f(x) ≈ e² + e²(x−1)

Maxima, Minima, and Saddle Points

Critical Point: A point x* where: f′(x*)=0

  • Slope = 0
  • Flat region

Linear Approximation at Critical Point: Lx*(f) = f(x*) — No linear change (constant).

Types of Critical Points

  1. Local Minimum: lowest point nearby, shape ∪.
  2. Local Maximum: highest point nearby, shape ∩.
  3. Saddle Point: neither max nor min, looks like a horse saddle.
    • Down in one direction, up in another.

Key Insight: Critical point ≠ always min/max; could be a saddle.

Importance in Machine Learning

  • ML is an optimization problem.
  • Goal: minimize loss function, find best parameters.

Why These Matter

  • Linear approximation → gradient descent.
  • Quadratic approximation → faster convergence (Newton methods).
  • Critical points → candidate solutions.

Quick Summary

  • Linear = first-order (slope).
  • Quadratic = includes curvature.
  • Taylor series = base idea.
  • Product & chain rules come from approximation.
  • Critical point: f′(x) = 0.
  • Types: minimum, maximum, saddle.

Lecture5: Multivariate Calculus: Lines and Planes in Higher Dimensional Space

Introduction

Idea

  • Single variable: f:R→R
  • Multiple variables: f:Rd→R

Input = vector, Output = single value.

Example: f(x₁, x₂) = x₁² + x₂²

  • Input: point in 2D
  • Output: real number

Intuition: Think of a landscape or surface; height = function value.

Geometry in Rd

A. Lines in Rd

  • Point + Direction Form: x = u + αv — Start at point u, move along direction v.
  • Two-Point Form: x = (1−α)u + αu′ — Line between two points.

Example: Line through (1,1) with direction (1,2): x = (1,1) + α(1,2)

B. Hyperplanes

  • Definition: WTx = b
  • Generalization of: Line (2D), Plane (3D)
  • Key Idea: W = normal vector (perpendicular), defines orientation.

Example in R³: x₁ + x₂ + x₃ = 1 — Plane with normal vector (1,1,1).

C. Points vs Vectors

Partial Derivatives

Idea: Differentiate with respect to one variable, keeping others constant.

Formula: ∂f/∂xi = limα→0 [ f(v+αei)−f(v) ] / α

Intuition: Move only along one axis direction.

Example: f(x₁, x₂) = x₁² + x₂² — ∂f/∂x₁ = 2x₁, ∂f/∂x₂ = 2x₂

Gradient

Definition: Gradient = vector of all partial derivatives: ∇f(x) = [∂f/∂x₁ , … , ∂f/∂xd]

Key Meaning

  • Gradient gives direction of maximum increase.
  • Measures rate of change in all directions.

Example

f(x₁, x₂) = x₁² + x₂² , ∇f = (2x₁, 2x₂)

Linear Function Case: f(x) = x₁ + 2x₂ + 3x₃, ∇f = (1,2,3) — Constant everywhere.

Quick Summary

  • Multivariate functions: many inputs, one output.
  • Line: x = u + αv.
  • Hyperplane: WTx = b.
  • Partial derivative: change in one direction.
  • Gradient: vector of all derivatives.
  • Gradient = direction of steepest ascent.

Lecture6: Multivariate Calculus: Linear Approximation and Applications

Linear Approximation (Multivariate)

1D Recall: f(x) ≈ f(x*) + f′(x*)(x−x*)

Multivariate Idea: For f:Rd→R, approximate near point v: Replace function with a plane (linear function).

General Formula: f(x) ≈ f(v) + ∇f(v)T(x−v)

Component Form: f(x) ≈ f(v) + Σi=1d ∂f/∂xi(v)(xi − vi)

Key Properties

  • Exact at x = v : f(v) = Lvf(v)
  • Valid only near v
  • Represents best linear fit locally

Geometric Interpretation of Gradient

A. Tangent Plane: Linear approximation forms a plane touching the surface at (v, f(v)). This is the tangent plane.

B. Contour (Level Set)

  • Level set: f(x) = f(v)
  • Gradient is perpendicular to contour.

Key Insight: Gradient ⊥ level curve, points in direction of maximum increase.

Directional Derivatives

Idea: Rate of change in a specific direction u.

Definition: Duf(v) = limα→0 [ f(v+αu) − f(v) ] / α

Computational Formula: Duf(v) = ∇f(v)Tu

Important Cases

  • Steepest Ascent: direction = ∇f(v), maximum increase.
  • Descent Direction: if ∇f(v)Tu < 0, function decreases.

Cauchy-Schwarz Insight: −‖a‖‖b‖ ≤ aTb ≤ ‖a‖‖b‖

  • Maximum when vectors aligned.
  • Minimum when opposite.

Quadratic Approximation (Advanced)

Formula: f(x) ≈ f(v) + ∇f(v)T(x−v) + 1/2 (x−v)T∇²f(v)(x−v)

Key Terms

  • ∇f(v): Gradient
  • ∇²f(v): Hessian matrix — Captures curvature.

Critical Points & Optimality

Critical Point: ∇f(v) = 0 — Flat region, no slope in any direction.

First-Order Condition: If v is min/max: ∇f(v) = 0.

Important: Every min/max is a critical point, but not every critical point is min/max (could be a saddle).

Worked Example

Problem: f(x₁, x₂) = x₁² + x₂², v = (6, 2)

Step 1: Function Value: f(6, 2) = 36 + 4 = 40

Step 2: Gradient: ∇f = (2x₁, 2x₂) — At v: ∇f(6, 2) = (12, 4)

Step 3: Apply Formula: f(x) ≈ 40 + 12(x₁ − 6) + 4(x₂ − 2)

Step 4: Simplify: f(x) ≈ 12x₁ + 4x₂ −40

Final Linear Approximation

Quick Summary

  • Multivariate approximation = tangent plane.
  • Gradient = direction of steepest increase, perpendicular to contours.
  • Directional derivative = ∇f·u.
  • Quadratic approximation uses Hessian (curvature).
  • Critical point: ∇f = 0.
