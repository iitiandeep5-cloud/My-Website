Week 11: Short Notes — Machine Learning Foundations

Lecture 1: Continuous Random Variables

Fundamental Concepts

Sample Space (Ω)

  • For continuous variables → uncountably infinite (e.g., all real numbers ℝ).
  • Unlike discrete → not countable.

Events (F) — Sigma Algebra

A valid event set must satisfy:

  • Ω is included.
  • If A is an event → Ac is also an event.
  • Countable union: A1, A2,... ⇒ ∪ Ai is also an event.

Random Variable (X)

  • A function: X : Ω → R
  • Continuous if it takes infinitely many values in an interval.

Source of Randomness

  • Complex systems (e.g., stock prices).
  • Uncertainty (e.g., waiting time, weather).

Probability Density Function (PDF)

Definition: Continuous version of PMF. Represents density, not direct probability.

Important Idea: P(X = x) = 0 — Exact value probability is always zero.

Key Properties

  • Non-negativity: fX(x) ≥ 0
  • Normalization: ∫−∞∞ fX(x) dx = 1

Important Notes

  • PDF can be greater than 1.
  • Units exist (e.g., per minute, per meter).

Cumulative Distribution Function (CDF)

Definition: Probability that X ≤ x: FX(x) = P(X ≤ x)

FX(x) = ∫−∞x fX(t) dt

Properties

  • FX(−∞) = 0.
  • FX(+∞) = 1.
  • Monotonically increasing.

Examples

Example A: Bus Waiting Time (Uniform Distribution)
Scenario: Bus every 15 minutes, arrival time is random.
Random Variable: X = waiting time ∈ [0, 15]

PDF: fX(x) = 1/15 for 0 ≤ x ≤ 15, else 0
CDF: FX(x) = 0 for x < 0; x/15 for 0 ≤ x ≤ 15; 1 for x > 15

Key Insight: Uniform distribution → CDF is linear.
Example B: Linear PDF
Given: fX(x) = x/2 for 0 ≤ x ≤ 2, else 0
Validation: Area = triangle = ½ × base × height = 1

CDF Calculation: FX(x) = ∫0x (t/2) dt = x²/4
Final CDF: 0 for x < 0; x²/4 for 0 ≤ x ≤ 2; 1 for x > 2

Key Insight: Linear PDF → Quadratic CDF.
Quick Revision
Continuous ⇒ P(X = x) = 0. | PDF area = 1. | CDF = integral of PDF.
Uniform PDF ⇒ linear CDF. | Linear PDF ⇒ quadratic CDF.

Lecture 2: Conditional PDF

Conditional Probability Density Function (PDF)

Idea

  • Describes distribution of X after knowing event A occurred.
  • Reduces uncertainty using given information.

Definition: Conditional density of X given A:

fX|A(x) = P(X ∈ [x, x+dx] ∣ A) / dx
Key Formula: fX|A(x) = fX(x) / P(A),   x ∈ A

Key Properties

  • fX|A(x) ≥ 0.
  • Total area = 1.
  • Outside event A → density = 0.
  • Inside A → scaled up by 1/P(A).

Intuition

  • Cut off invalid region.
  • Stretch remaining density to make total area = 1.
Example: Conditional PDF
Given: fX(x) = x/2 for 0 ≤ x ≤ 2, else 0. Event: A = {X > 1}.

Step 1: Find P(A): P(A) = ∫12 (x/2) dx = 3/4
Step 2: Conditional PDF: For 1 ≤ x ≤ 2: fX|A(x) = (x/2)/(3/4) = 2x/3
Final Answer: fX|A(x) = 2x/3 for 1 ≤ x ≤ 2, else 0

Insight: Density increases (scaled up). Values outside A vanish.

Functions of Random Variables

Idea: If Y = g(X), then Y is also a random variable.

Goal: Find PDF of Y, denoted fY(y).

General Principle: Probability is preserved: P(y ≤ Y ≤ y + dy). Transformation changes: range, and density (must still integrate to 1).

Example 1: Linear Transformation (Y = X/2)
Given: X ~ Uniform[−1,1]. fX(x) = ½, −1 ≤ x ≤ 1

Step 1: Range: Y ∈ [−1/2, 1/2]
Step 2: PDF: fY(y) = fX(2y)·2 = (1/2)·2 = 1
Final Answer: fY(y) = 1 for −1/2 ≤ y ≤ 1/2, else 0

Insight: Uniform stays uniform, density adjusts to new interval size.
Example 2: Absolute Value Transformation (Y = |X|)
Given: X ~ Uniform[−1,1]. fX(x) = 1/2, −1 ≤ x ≤ 1

Step 1: Range: Y ∈ [0,1]
Step 2: Key Idea: Two values map to one: x and −x → same y.
Result: fY(y) = 2·fX(y) = 2·1/2 = 1
Final Answer: fY(y) = 1 for 0 ≤ y ≤ 1, else 0

Insight: Folding effect (negative side overlaps positive). Density doubles.
Quick Revision
Conditional PDF: Cut + scale: fX|A(x) = fX(x)/P(A)
Transformations: Linear → scale density by derivative. Many-to-one (like |X|) → add densities.
Key Tricks: Always adjust range first. Ensure total area = 1. Think: "Where did probability mass go?"

Lecture 3: Expectation

Expectation (Mean) of Continuous Random Variable

Definition: Continuous version of average. Replace summation with integration:

E[X] = ∫−∞∞ x fX(x) dx

Intuition: Acts like a balance point (center of mass) where the probability "weight" balances.

Example 1: Uniform Distribution X ~ U(a,b)
PDF: fX(x) = 1/(b−a), a ≤ x ≤ b
Result: E[X] = (a+b)/2
Insight: Mean = midpoint of interval.

Example 2: Triangular Density
Given: fX(x) = x/2, 0 ≤ x ≤ 2
Result: E[X] = ¾
Insight: Skewed right → mean shifts right.

Properties of Expectation

  • Linearity: E[X+Y] = E[X] + E[Y] — Works always (no independence needed).
  • Expectation of Function: E[g(X)] = ∫−∞∞ g(x)fX(x) dx — No need to find PDF of Y = g(X).

Variance

Definition: Measures spread around mean.

Standard Formula: Var(X) = E[(X − E[X])²]
Shortcut Formula: Var(X) = E[X²] − (E[X])²
Standard Deviation: σ = √Var(X)

Properties of Variance

  • Var(X) ≥ 0.
  • Var(constant) = 0.
  • Scaling: Var(aX) = a²Var(X)
  • Not linear: Var(X+Y) ≠ Var(X) + Var(Y) (in general)
Example: Uniform Distribution X ~ U(a, b)
Result: Var(X) = (b−a)²/12
Insight: Larger interval → larger spread.

Conditional Expectation

Definition: Expected value after knowing event A occurred:

E[X|A] = ∫−∞∞ x fX|A(x) dx

Law of Total Expectation: E[X] = E[X|A]P(A) + E[X|Ac]P(Ac)

Idea: Break problem into cases, take weighted average.

Example: Bus Waiting Time
Scenario: Arrive between 7:10–7:30. Bus at 7:15 and 7:30.

Case 1: A (before 7:15) — P(A) = ¼. Waiting time: Uniform [0,5]. E[X|A] = 2.5.
Case 2: Ac (after 7:15) — P(Ac) = ¾. Waiting time: Uniform [0,15]. E[X|Ac] = 7.5.

Final Answer: E[X] = (2.5)(¼) + (7.5)(¾) = 6.25
Insight: Split into simple cases, combine using probabilities.
Quick Revision Cheatsheet
Expectation: Mean = balance point. Uniform → midpoint.
Variance: Spread = "how far from mean." Shortcut: E[X²] − (E[X])²
Conditional Expectation: Use conditional PDF or split cases: E[X] = ∑E[X|case]·P(case)
Key Tricks: Always check range first. Use symmetry when possible. Break problems into simpler parts.

Lecture 4: Multiple Random Variables

Joint Density & Joint CDF

Joint PDF: Describes probability of X and Y together. Probability over a small rectangle:

fXY(x,y) = P(x ≤ X ≤ x+dx, y ≤ Y ≤ y+dy) / (dx·dy)

Intuition: Think of a 3D surface — height = density, area = probability.

Joint CDF: FXY(x,y) = P(X ≤ x, Y ≤ y)

Key Properties

  • Non-negative: fXY(x, y) ≥ 0.
  • Total probability = 1: ∫∫ fXY(x,y) dx dy = 1
  • Limits: F(−∞,−∞) = 0, F(∞,∞) = 1.
  • Monotonic: Increasing in both x and y.

Marginal Densities

Idea: Get distribution of one variable by removing the other.

fX(x) = ∫−∞∞ fXY(x,y) dy
fY(y) = ∫−∞∞ fXY(x,y) dx

Trick: Integrate out the other variable.

Conditional Densities

Definition: Distribution of one variable given the other.

For X given Y: fX|Y(x|y) = fXY(x,y) / fY(y)
For Y given X: fY|X(y|x) = fXY(x,y) / fX(x)

Interpretation: Fix one variable → study the other. Example: Given height → distribution of weight.

Independence

Definition: fXY(x, y) = fX(x)·fY(y)

Equivalent (Expectation Form): E[g(X)h(Y)] = E[g(X)]·E[h(Y)]

Intuition: Knowing X gives no information about Y.

Important: If joint ≠ product → dependent.

Example: Height (X) & Weight (Y)
Given: Height: 1 < x < 2. Weight: 50x − 20 ≤ y ≤ 50x + 20. Joint PDF = constant C.

Step 1: Find C: Region is a parallelogram. Area = 40. C·40 = 1 ⇒ C = 1/40
Step 2: Marginal of X: fX(x) = ∫50x−2050x+20 (1/40) dy = 1 ⇒ Uniform distribution
Step 3: Are X and Y Independent? No (Dependent).

Reason: Some combinations impossible (e.g., tall height + very low weight). But marginals may still be non-zero.

Key Insight: Independence requires: all combinations possible, and joint = product of marginals.
Quick Revision
Joint PDF: 2D density → integrates to 1.
Marginals: Integrate out one variable.
Conditional Density: fX|Y(x|y) = fXY(x,y)/fY(y)
Independence: Joint = product of marginals.

Key Tricks: Always visualize region. Check valid combinations. Use integration carefully.

Lecture 5: Independent Random Variables

Sum of Independent Random Variables

Idea: If Z = X+Y, its PDF is obtained using convolution.

fZ(z) = ∫−∞∞ fX(x)fY(z−x) dx

Intuition: Slide one PDF over another and compute overlap.

Example: X,Y ~ Uniform(0,1)
Result (Triangular Distribution): fZ(z) = z for 0 ≤ z ≤ 1; 2−z for 1 < z ≤ 2; 0 otherwise
Insight: Shape = triangle, peak at z = 1.

Maximum of Independent Variables

Idea: If Z = max(X,Y), both must be ≤ z.

CDF Formula: FZ(z) = FX(z)·FY(z)
PDF: fZ(z) = d/dz FZ(z)
Example: X,Y ~ Uniform(0,1): FZ(z) = z², fZ(z) = 2z, 0 ≤ z ≤ 1
Insight: More density near 1 → larger values more likely.

Minimum of Independent Variables

Idea: If Z = min(X,Y), at least one must be ≤ z.

CDF Formula: FZ(z) = 1 − (1 − FX(z))(1 − FY(z))

Insight: Opposite of maximum → more density near 0.

Covariance & Correlation

Covariance: Cov(X,Y) = E[XY] − E[X]E[Y]

Meaning: Measures linear relationship.

  • Positive → move together.
  • Negative → opposite direction.
Correlation Coefficient: ρ(X,Y) = Cov(X,Y) / √(Var(X)·Var(Y))

Range: −1 ≤ ρ ≤ 1.

Key Concepts

  • Independence ⇒ Cov(X,Y) = 0.
  • But Cov(X,Y) = 0 does not imply independence.
Example: Uncorrelated but Dependent
X ~ U(−1, 1). Y = X². Result: Cov(X,Y) = 0. But Y depends on X → dependent.

Insight: Covariance captures only linear relation.

Random Vectors & Covariance Matrix

Random Vector: (X1, X2, ..., Xn)

Covariance Matrix: Σ =
[ Var(X1)   Cov(X1,X2)   ... ]
[ Cov(X2,X1)   Var(X2)   ... ]
[ ...   ...   ⋱ ]

Properties

  • Diagonal = variances.
  • Off-diagonal = covariances.
  • Symmetric: Cov(X,Y) = Cov(Y,X).

Insight: Captures relationships between all variables together.

Quick Revision
Sum: Use convolution. Uniform + Uniform → triangle.
Max: FZ = FX·FY | Min: FZ = 1 − (1−FX)(1−FY)
Covariance: Measures linear relation. Independence ⇒ zero covariance.
Correlation: Normalized covariance (−1 to 1).
Matrix: Diagonal → variance, symmetric structure.
Final Tips
Always check independence first. For sums → think convolution. For max/min → use CDF approach. Don't confuse: uncorrelated ≠ independent.

Lecture 6: Transformed Random Variables

Core Concept

  • Idea: Start with two variables W, X with known joint PDF.
  • Define new variables: Y = G(W, X), Z = H(W, X)
  • Goal: Find joint PDF of (Y, Z).

Requirement: Invertibility

Why needed? Must recover original variables from new ones. Condition: there exist inverse functions:

W = q(y,z), X = r(y,z)

Important: If not invertible → information lost.

Example (Not invertible): Y = W, Z = W ⇒ cannot recover X

Change of Variables Rule

Intuition

  • Small rectangle in (Y, Z) ↔ parallelogram in (W, X).
  • Probability remains same, only area changes.

Key Formula

Joint Density Transformation: fYZ(y, z) = fWX(q(y,z), r(y,z))·|det(J)|

Jacobian Matrix: J = [ ∂W/∂y   ∂W/∂z ; ∂X/∂y   ∂X/∂z ]
Determinant: det(J) = (∂W/∂y)(∂X/∂z) − (∂W/∂z)(∂X/∂y)

Absolute Value: Use |det(J)| because area cannot be negative.

Step-by-Step Method

  1. Define transformation Y, Z.
  2. Check invertibility.
  3. Find inverse functions W, X.
  4. Compute Jacobian.
  5. Substitute into formula.
  6. Determine valid region.
Example: Y = W + X, Z = W − X
Given: W, X ~ Uniform(0,1), independent. fWX(w,x) = 1

Step 1: Inverse Transformation — W = (y+z)/2, X = (y−z)/2
Step 2: Jacobian — J = [½   ½ ; ½   −½], det(J) = (½·−½) − (½·½) = −½, |det(J)| = ½
Step 3: Joint PDF — fYZ(y,z) = fWX((y+z)/2,(y−z)/2)·½ = ½ (since fWX = 1)
Step 4: Region (Support) — From 0 ≤ (y+z)/2 ≤ 1 and 0 ≤ (y−z)/2 ≤ 1: forms a diamond-shaped region.

Final Result: fYZ(y,z) = ½ inside region, 0 outside
Are Y and Z Independent? No. Reason: Restricted region (not rectangular). Some (y,z) pairs impossible.
Quick Revision
Transformation Formula: fYZ = fWX·|det(J)|
Steps: Find inverse → Compute Jacobian → Apply formula → Define region.
Key Insights: Jacobian = area scaling factor. Invertibility is essential. Region determines dependence.
Common Mistakes: Forgetting absolute value. Ignoring region constraints. Assuming independence.
Exam Tips
Always check invertibility. Write inverse clearly. Draw region (very helpful). Shortcut: If region not rectangular → dependent.

Lecture 7: Uniform, Exponential, Normal

Uniform Distribution

Notation: X ~ Uniform(a, b)

PDF: fX(x) = 1/(b−a), a ≤ x ≤ b

Idea: All values in [a, b] are equally likely.

Mean: E[X] = (a+b)/2   Variance: Var(X) = (b−a)²/12

Example: Waiting time uniformly between 0–10 minutes → Mean = 5 minutes.

Exponential Distribution

Notation: X ~ Exponential(λ)

PDF: fX(x) = λe−λx, x ≥ 0
CDF: FX(x) = 1 − e−λx

Mean: E[X] = 1/λ

Key Idea: Models waiting time until an event occurs.

Memoryless Property: P(X ≥ a | X ≥ b) = P(X ≥ a−b)

Meaning: Past does not affect future.

Minimum of Exponentials: If X ~ Exp(λ), Y ~ Exp(τ): min(X,Y) ~ Exp(λ+τ)

Applications

  • System failure (multiple components).
  • Time to next event.

Examples: Bulb lifetime, laptop fails if CPU or RAM fails.

Normal (Gaussian) Distribution

Standard Normal: Z ~ N(0, 1)

PDF: fZ(z) = (1/√(2π)) e−z²/2

Properties

  • Mean = 0.
  • Variance = 1.
  • Symmetric about 0.

General Normal: X ~ N(μ, σ²)

Transformation: X = σZ + μ

PDF: fX(x) = (1/(σ√(2π))) e−½((x−μ)/σ)²

Parameters

  • Mean: E[X] = μ.
  • Variance: Var(X) = σ².

Key Concepts

  • Central Limit Theorem (CLT): Sum of many random variables → Normal distribution.
  • Real-Life Examples: Measurement noise, heights, marks, errors.
  • Symmetry: Bell-shaped curve, equal spread on both sides.
Quick Revision Cheatsheet
Uniform: Flat distribution, mean = midpoint.
Exponential: Waiting time, memoryless property.
Normal: Bell curve, most natural phenomena.
Final Tips
Uniform → simplest distribution. Exponential → only memoryless distribution. Normal → appears everywhere (CLT).

Common Mistakes: Forgetting domain (especially exponential: x ≥ 0). Mixing up mean/variance formulas. Ignoring symmetry in normal distribution.
