Lecture 1: Complex Matrices

Complex Vector Space ($\mathbb{C}^n$)

Definition: $\mathbb{C}^n$ is like $\mathbb{R}^n$, but each component is a complex number.
Vector: $x = [x_1, x_2, \ldots, x_n]$, where $x_i = a + ib$

Basic Operations

  • Addition: $(a+ib) + (c+id) = (a+c) + i(b+d)$
  • Multiplication: $(a+ib)(c+id) = (ac-bd) + i(bc+ad)$
  • Complex Conjugate: $\overline{(a+ib)} = a - ib$
  • Polar Form: $z = re^{i\theta}$,   $\bar{z} = re^{-i\theta}$

Linear Combination: Scalars can be complex: $\sum c_i x_i$

Inner Product & Length

Problem with Real Dot Product: $x^T y$ can give wrong results.
Example: $x = [1, i]^T \Rightarrow 1^2 + i^2 = 1 - 1 = 0$ ✘

Correct Inner Product: $\langle x, y \rangle = x^* y = (\bar{x})^T y$

Properties

  • Conjugate symmetry: $\langle x, y \rangle = \overline{\langle y, x \rangle}$
  • Scalar rules:
    • $\langle x, cy \rangle = c \langle x, y \rangle$
    • $\langle cx, y \rangle = \bar{c} \langle x, y \rangle$

Length (Norm): $\|x\| = \sqrt{x^* x}$

Example

For $x = [1, i]^T$:
$\|x\|^2 = (\bar{1})(1) + (\bar{i})(i) = 1 + 1 = 2$

Conjugate Transpose ($A^*$)

Definition: $A^* = (\bar{A})^T$ — take complex conjugate + transpose

Example

$$A = \begin{bmatrix} 1+i & 2-4i \\ 3-2i & i \end{bmatrix}$$ $$A^* = \begin{bmatrix} 1-i & 2+4i \\ 3+2i & -i \end{bmatrix}$$

Properties

  • $(A^*)^* = A$
  • $(AB)^* = B^* A^*$
  • For real matrices: $A^* = A^T$

Hermitian Matrices

Definition: $A^* = A$ — the complex version of a symmetric matrix.

  • Key Points:
    • Diagonal elements are real
    • Important in: Linear Algebra, Machine Learning, Quantum Mechanics

Why Important?

  • Hermitian matrices are diagonalizable
  • They have real eigenvalues

Quick Summary

  • $\mathbb{C}^n \to$ vectors with complex numbers
  • Inner product uses conjugate
  • Length → $\sqrt{x^* x}$
  • Conjugate transpose → flip + conjugate
  • Hermitian matrix → $A^* = A$

Lecture 2: Hermitian Matrices

Definition & Basics

A square matrix $A$ is Hermitian if: $A = A^*$, where $A^*$ = conjugate transpose of $A$

Conjugate Transpose

  • Take complex conjugate of each element
  • Then transpose the matrix: $A^* = (\bar{A})^T$

Important Points

  • Diagonal elements are always real
  • Off-diagonal elements are complex conjugates: $a_{ij} = \bar{a}_{ji}$

Special Case

  • Every real symmetric matrix is Hermitian (because conjugate = same number)

Key Properties

Property 1: Real Eigenvalues

All eigenvalues of a Hermitian matrix are real.
Key idea: $Ax = \lambda x \Rightarrow \lambda = \bar{\lambda} \Rightarrow \lambda$ is real

Property 2: Orthogonal Eigenvectors

Eigenvectors corresponding to distinct eigenvalues are orthogonal.
Orthogonality condition: $x^* y = 0$
Result: $(\lambda_1 - \lambda_2)(x^* y) = 0 \Rightarrow x^* y = 0$ (for $\lambda_1 \neq \lambda_2$)

Diagonalization

  • Key Idea: Hermitian matrices are always diagonalizable
  • Formula: $A = PDP^{-1}$
    • $P$: matrix of eigenvectors
    • $D$: diagonal matrix of eigenvalues
  • Important Notes:
    • Eigenvectors are orthogonal ⇒ linearly independent
    • Even with repeated eigenvalues, diagonalization is possible

Standard Example

Given Matrix: $$A = \begin{bmatrix} 2 & 3+3i \\ 3-3i & 5 \end{bmatrix}$$ Step 1: Check Hermitian — conjugate transpose gives the same matrix, so $A$ is Hermitian.

Step 2: Eigenvalues
$\det(A - \lambda I) = 0$
$(2-\lambda)(5-\lambda) - (3-3i)(3+3i) = 0$
$= (2-\lambda)(5-\lambda) - (9+9)$
$= \lambda^2 - 7\lambda - 8 = 0$
$\Rightarrow (\lambda-8)(\lambda+1) = 0$
Eigenvalues: $\lambda_1 = 8$, $\lambda_2 = -1$

Step 3: Eigenvectors
For $\lambda_1 = 8$: $x = [1,\ 1+i]$
For $\lambda_2 = -1$: $y = [1-i,\ -1]$

Step 4: Orthogonality Check
$x^* y = 0$ — hence eigenvectors are orthogonal.

Quick Summary

  • $A = A^* \to$ Hermitian
  • Diagonal elements → Real
  • Eigenvalues → Always real
  • Eigenvectors → Orthogonal
  • Always → Diagonalizable

Lecture 3: Unitary Matrices

Definition & Core Concepts

A square matrix $U$ is Unitary if: $U^* U = I$

  • $U^*$ = conjugate transpose of $U$
  • $I$ = identity matrix

Key Meaning

  • Columns (and rows) of $U$ are orthonormal
  • It is the complex version of orthogonal matrices
Important Result: $U^{-1} = U^*$ — so, inverse = conjugate transpose

Examples

Example 1 (Standard Check)

To verify if a matrix is unitary:
  1. Find $U^*$
  2. Check: $U^* U = I$

Example 2 (Trigonometric Matrix)

$$U = \begin{bmatrix} \cos t & \sin t \\ -\sin t & \cos t \end{bmatrix}$$ Compute $U^*$ (transpose here, since real): $$U^T = \begin{bmatrix} \cos t & -\sin t \\ \sin t & \cos t \end{bmatrix}$$ $U^T U = I$ — hence, $U$ is unitary (orthogonal in the real case).

Key Properties

Property 1: Length Preservation (Isometry)

Unitary matrices do not change vector length:
$\|Ux\| = \|x\|$
Also preserves inner product: $(Ux)^*(Uy) = x^* y$

Property 2: Eigenvalues

Eigenvalues of a unitary matrix satisfy: $|\lambda| = 1$
Meaning: They lie on the unit circle in the complex plane. Not necessarily real.

Property 3: Orthogonal Eigenvectors

Eigenvectors corresponding to distinct eigenvalues are orthogonal: $x^* y = 0$ (for $\lambda_1 \neq \lambda_2$)

Diagonalization & Spectral Theorem

Spectral Theorem (Important): For a Hermitian matrix $A$: $A = U \Lambda U^*$
  • $U$: unitary matrix (eigenvectors)
  • $\Lambda$: diagonal matrix (eigenvalues)
Real Case (Special Form): $A = Q \Lambda Q^T$, where $Q$: orthogonal matrix

Key Insight: Unitary matrices help in diagonalizing Hermitian matrices. This simplifies many problems in:

  • Quantum mechanics
  • Linear algebra
  • Signal processing

Quick Summary

  • $U^* U = I \to$ Unitary
  • $U^{-1} = U^*$
  • Length preserved → $\|Ux\| = \|x\|$
  • Eigenvalues → $|\lambda| = 1$
  • Eigenvectors → Orthogonal
  • Used in → Spectral Theorem & Diagonalization

Lecture 4: Diagonalization of Hermitian Matrices (Part 1)

Fundamental Definitions

  • Conjugate Transpose: $A^* = (\bar{A})^T$ — take complex conjugate + transpose
  • Hermitian Matrix: $A = A^*$
    • Diagonal elements → real
    • Special class of matrices
  • Unitary Matrix: $U^* U = I$ — important: $U^{-1} = U^*$
  • Unitarily Diagonalizable: A matrix $A$ is unitarily diagonalizable if $A = U \Lambda U^*$, where $U$: unitary matrix, $\Lambda$: diagonal (eigenvalues)
  • Upper Triangularizable: $A = UTU^*$, where $T$: upper triangular matrix

Schur's Theorem

Statement: Every square complex matrix can be written as: $A = UTU^*$, where $U$: unitary, $T$: upper triangular

Key Implications:

  • $A$ and $T$ have the same eigenvalues
  • Works for all complex matrices

Important Insight:

  • If $A$ is Hermitian, then $T$ becomes diagonal — hence $A = U \Lambda U^*$
  • This proves Hermitian matrices are unitarily diagonalizable

Procedure (Schur Algorithm)

Step-by-Step

Step 1: Find Eigenvalue & Eigenvector
  • Solve: $\det(A - \lambda I) = 0$
  • Get eigenpair $(\lambda_1, Z_1)$
Step 2: Normalize — convert $Z_1$ into a unit vector

Step 3: Extend Basis
  • Use Gram-Schmidt
  • Form orthonormal basis of $\mathbb{C}^n$
Step 4: Form Unitary Matrix $U_1$ — columns = orthonormal basis vectors

Step 5: Transform Matrix $$U_1^* A U_1 = \begin{bmatrix} \lambda_1 & * \\ 0 & B \end{bmatrix}$$ where $B$: smaller matrix

Step 6: Repeat (Recursion) — apply same process to $B$

Step 7: Final Matrix — $A = UTU^*$

Worked Example (3×3 Matrix)

Given: $$A = \begin{bmatrix} 5 & 8 & 16 \\ 5 & 0 & 9 \\ -3 & -5 & -10 \end{bmatrix}$$ Step 1: Eigenvalues: $\lambda_1 = 1$, $\lambda_2 = -3$

Step 2: Eigenvector for $\lambda_1 = 1$: $Z_1 = \begin{bmatrix} -2 \\ -1 \\ 1 \end{bmatrix}$

Step 3: Normalize: $\|Z_1\| = \sqrt{6} \Rightarrow \dfrac{1}{\sqrt{6}}\begin{bmatrix} -2 \\ -1 \\ 1 \end{bmatrix}$

Step 4: Construct $U_1$: complete orthonormal basis using Gram-Schmidt

Step 5: Transform: $$U_1^* A U_1 = \begin{bmatrix} 1 & * & * \\ 0 & B & * \\ 0 & 0 & * \end{bmatrix}$$ where $B$ is $2 \times 2$

Step 6: Process $B$ — eigenvalue: $-3$; find eigenvector → form matrix $P$

Final Result: $A = UTU^*$ — matrix is upper triangularized

Important Conclusion

  • Matrix has repeated eigenvalues
  • Not enough independent eigenvectors
  1. Not diagonalizable
  2. But always triangularizable (Schur)

Quick Summary

  • $A = UTU^* \to$ always possible (Schur)
  • $T \to$ upper triangular
  • Eigenvalues → diagonal of $T$
  • Hermitian case → $T = \Lambda$ (diagonal)
  • $\Rightarrow$ Hermitian $\Rightarrow$ unitarily diagonalizable

Lecture 5: Diagonalization of Hermitian Matrices (Part 2)

Spectral Theorem (Main Idea)

  • The Spectral Theorem tells us when a matrix can be diagonalized using a unitary/orthogonal matrix.
Key Statement: Every Hermitian matrix is unitarily diagonalizable.
Meaning: $U^* A U = D$
$U$: Unitary matrix, $D$: Diagonal matrix (contains eigenvalues)

Important:

  • Hermitian ⇒ Unitarily diagonalizable ✅
  • Unitarily diagonalizable ⇏ Hermitian ❌ (not always true)

Why It Works (Proof Idea – Intuition)

  • By Schur's Theorem: $U^* A U = T$ (upper triangular)
  • If $A = A^*$ (Hermitian), then: $T = T^*$
  • But:
    • $T$ is upper triangular
    • $T^*$ is lower triangular
  • Only possible if $T$ is diagonal — hence proved!

Important Definitions & Formulae

  • Hermitian Matrix: $A = A^*$
  • Unitary Matrix: $U^* = U^{-1}$, $U^* U = I$
  • Unitary Diagonalization: $U^* A U = D$
  • Real Eigenvalues: Eigenvalues of Hermitian matrices are always real.
  • Normalization: To form $U$, eigenvectors must be orthonormal: $\|v\| = 1$

Real Case (Symmetric Matrices)

Theorem: If $A = A^T$, then: $Q^T A Q = D$
  • $Q$: Orthogonal matrix
  • $Q^T Q = I$

Insight:

  • Eigenvalues are real
  • Eigenvectors can be chosen real + orthonormal

Examples

Example 1: Hermitian Matrix

  • Eigenvalues: $\lambda_1 = 1$, $\lambda_2 = 4$
  • Eigenvectors: $Z_1 = [-1+i,\ 1]$, $Z_2 = [1-i,\ 2]$
  • After normalization: $U_1 = \frac{1}{\sqrt{3}}[-1+i,\ 1]$, $U_2 = \frac{1}{\sqrt{6}}[1-i,\ 2]$
  • Final result: $U^* A U = \begin{bmatrix} 1 & 0 \\ 0 & 4 \end{bmatrix}$ — diagonal with real eigenvalues

Example 2: Not Hermitian but Diagonalizable

Matrix: $A = \begin{bmatrix} 0 & 1 \\ -1 & 0 \end{bmatrix}$ — not Hermitian: $A^* \neq A$
Eigenvalues: $\lambda_1 = i$, $\lambda_2 = -i$
Eigenvectors: $Z_1 = [1, -i]$, $Z_2 = [1, i]$
Result: $U^* A U = \begin{bmatrix} i & 0 \\ 0 & -i \end{bmatrix}$
✔️ Diagonalizable   ❌ Not Hermitian (eigenvalues not real)

Quick Summary

  • Hermitian ⇒ Always diagonalizable using unitary matrix
  • Eigenvalues of Hermitian matrices are real
  • Real symmetric matrices ⇒ orthogonal diagonalization
  • Not all diagonalizable matrices are Hermitian
