About This Subject

Machine Learning Techniques (MLT) is the applied counterpart to Machine Learning Foundations. While MLF builds the mathematical base, MLT focuses on the actual algorithms and techniques used to solve real-world problems. This subject covers both unsupervised and supervised learning methods in depth.

From clustering and dimensionality reduction to regression, classification, support vector machines, and neural networks — this subject equips you with the full toolkit of a modern machine learning engineer. You will learn not just how these algorithms work, but when and why to use each one depending on your data and problem type.

These week-wise notes are designed for students of IIT Madras BS in Data Science, NPTEL machine learning courses, BCA, MCA, and B.Tech programs who want to master the practical techniques used in data science and AI roles at top companies.

Week 1 – Introduction; Unsupervised Learning - PCA

Introduction to machine learning techniques and overview of the course. Deep dive into Principal Component Analysis (PCA) — how it reduces the number of features while retaining maximum variance. Applications of PCA in data visualization, noise reduction, and speeding up downstream machine learning algorithms.

View

Week 2 – Unsupervised Learning - Kernel PCA

Understanding the limitations of linear PCA and how Kernel PCA extends it to handle non-linear data. The kernel trick — how it maps data into higher dimensional spaces without explicitly computing the transformation. Comparison of different kernel functions and when to use each one.

View

Week 3 – Unsupervised Learning - Clustering - K-Means and Kernel K-Means

Understanding clustering as an unsupervised technique to group similar data points together. The K-Means algorithm — initialization, assignment, and update steps. Choosing the right number of clusters using the elbow method. Kernel K-Means for handling non-linearly separable clusters with practical examples.

View

Week 4 – Unsupervised Learning - Gaussian Mixture Models and EM Algorithm

Recap of Maximum Likelihood Estimation and Bayesian estimation. Gaussian Mixture Models (GMM) as a soft clustering approach where each data point can belong to multiple clusters with different probabilities. The Expectation Maximization (EM) algorithm for fitting GMMs and its convergence properties.

View

Week 5 – Supervised Learning - Regression - Least Squares and Bayesian View

Introduction to supervised learning with regression. Ordinary least squares regression — deriving the closed-form solution and understanding its assumptions. The Bayesian view of regression — treating model parameters as distributions rather than fixed values, leading to more robust predictions and uncertainty estimates.

View

Week 6 – Supervised Learning - Ridge and LASSO Regression

The problem of overfitting in regression and how regularization solves it. Ridge regression (L2 regularization) — shrinking coefficients to prevent overfitting. LASSO regression (L1 regularization) — performing automatic feature selection by driving some coefficients to exactly zero. Choosing between Ridge and LASSO for different problems.

View

Week 7 – Supervised Learning - Classification - K-NN and Decision Trees

K-Nearest Neighbours (K-NN) — the simplest classification algorithm and how the choice of K affects bias-variance tradeoff. Decision Trees — how they split data using information gain and Gini impurity. Understanding tree depth, pruning, and how decision trees handle both classification and regression tasks.

View

Week 8 – Supervised Learning - Naive Bayes Classifier

Generative models vs discriminative models. The Naive Bayes classifier — applying Bayes theorem with the naive assumption of feature independence. Gaussian Naive Bayes, Multinomial Naive Bayes, and Bernoulli Naive Bayes. Why Naive Bayes works surprisingly well in text classification and spam detection despite its simplifying assumptions.

View

Week 9 – Discriminative Models - Perceptron and Logistic Regression

The Perceptron — the oldest neural network unit and how it learns a linear decision boundary. Logistic regression as a probabilistic classifier — using the sigmoid function to output class probabilities. Training logistic regression with gradient descent and understanding the cross-entropy loss function used in modern deep learning.

View

Week 10 – Support Vector Machines

Support Vector Machines (SVM) — finding the maximum margin hyperplane that best separates classes. Hard margin vs soft margin SVM for handling non-linearly separable data. The kernel trick applied to SVMs for non-linear classification. Why SVMs are still widely used in industry for text classification, bioinformatics, and image recognition.

View

Week 11 – Ensemble Methods - Bagging and Boosting

Ensemble methods — combining multiple weak models to build a stronger one. Bagging (Bootstrap Aggregating) and Random Forests — how training on random subsets of data reduces variance. Boosting and AdaBoost — sequentially training models where each new model focuses on the mistakes of the previous one. Why ensemble methods dominate machine learning competitions.

View

Week 12 – Artificial Neural Networks and Multiclass Classification

Introduction to artificial neural networks — layers, neurons, activation functions, and forward propagation. Backpropagation algorithm for training neural networks. Multiclass classification using softmax activation and categorical cross-entropy loss. How neural networks form the foundation of modern deep learning and AI systems used in image recognition, natural language processing, and more.

View

Why Study Machine Learning Techniques?

Machine learning is transforming every industry — from healthcare and finance to e-commerce and entertainment. Understanding the core techniques of machine learning gives you the ability to build systems that learn from data and make intelligent predictions.

The algorithms covered in this subject — regression, classification, clustering, SVMs, and neural networks — are the exact tools used by data scientists and ML engineers at companies like Google, Amazon, Netflix, and thousands of startups worldwide. Mastering these techniques opens doors to some of the highest-paying and most exciting careers in technology.

These notes are ideal for students of IIT Madras BS in Data Science, NPTEL machine learning courses, and anyone preparing for data science or machine learning engineer roles. Combined with the MLF course, you will have a complete and deep understanding of modern machine learning.
